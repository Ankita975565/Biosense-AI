# Biosense-AI
AI - powered preventive health monitoring system that provides health analysis, risk assessment, health scores, and personalized wellness recommendations.

## Features
- Health Score Calculation
- BMI Analysis
- Health Risk Assessment
- Personalized Wellness Recommendations
- Future IoT & Wearable Integration

## Tech Stack
- Frontend: Streamlit 
- Backend: Python 
- Database: AWS DynamoDB (planned integration)
- AI: Scikit-learn, Pandas, NumPy
- Development Tools: GitHub
